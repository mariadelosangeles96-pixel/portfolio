# Jardín de código - Portfolio personal
### Mars Gómez (1º DAW)

Este es mi proyecto de portfolio. 
El concepto une dos de mis pasiones: el desarrollo de software y el arte digital.
Idea estética: Cottagecore Digital. Una mezcla de interfaces de sistemas operativos retro con elementos orgánicos, botánicos y dibujos realizados a mano con tableta Wacom.

## Tecnologías utilizadas hasta el momento
* **HTML5:** Estructura semántica para accesibilidad y SEO.
* **CSS3:** Diseño personalizado.
* **JavaScript:** Funcionalidad.
* **Photoshop con Wacom Intuos:** Ilustraciones y elementos visuales.
